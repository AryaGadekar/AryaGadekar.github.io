## Selected projects (click each project folder for code, data snippets, and dashboards)

### 1) Sales Forecasting & Inventory Optimization  
Tools: SQL, Python (Pandas), Prophet/ARIMA, Power BI  
What I did: End-to-end pipeline from raw sales to forecast, created daily dashboards for stock re-order suggestions and a simple model to predict 30-day demand.  
Impact: Reduced stockouts for the demo dataset and produced a reorder policy that cut projected inventory holding by 12% while maintaining service levels.

### 2) Customer Segmentation & Churn Analysis  
Tools: SQL, Python (scikit-learn), Power BI  
What I did: Cleaned transactional and engagement data, defined behavioral segments, built a churn model, and translated outputs into retention playbooks and dashboard visualizations.  
Impact: Prioritized top-20% of customers for targeted campaigns based on LTV estimates and created an intuitive KPI dashboard for monthly executive reviews.

### 3) Sentiment Mining for Product Feedback  
Tools: Python (NLP), Tableau / Power BI  
What I did: Scraped and processed product reviews, performed sentiment and topic modeling, surfaced recurring issues, and recommended product and communications fixes.  
Impact: Delivered prioritized product fixes and a weekly dashboard that non-technical product managers used to track sentiment trends.

1. Fork or clone.  
2. Open `index.html` and update the header links and project card links.  
3. Put images and dashboards in `assets/images` and `dashboards/`.  
4. Name the repo `your-username.github.io` and enable GitHub Pages in settings. Your site will be live at `https://your-username.github.io`.
